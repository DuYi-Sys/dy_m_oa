HTML 5 <input> 标签

定义和用法

<input> 标签定义输入字段，用户可在其中输入数据。

HTML 4.01 与 HTML 5 之间的差异

在 HTML 5 中，该元素有很多新属性，同时不再支持 HTML 4.01 中的一些属性。

在 HTML 5 中，type 属性有很多新的值。
