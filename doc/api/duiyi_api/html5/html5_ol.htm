HTML 5 <ol> 标签

定义和用法

<ol> 标签定义有序列表。

HTML 4.01 与 HTML 5 之间的差异

在 HTML 4.01 中，不赞成使用 "start" 属性，在 HTML 5 中是允许的。

在 HTML 4.01 中，不赞成使用 "compact" 和 "type" 属性，在 HTML 5 中，不再支持这两个属性。

提示和注释

提示：请使用 CSS 来定义列表的类型。

例子

<ol>
   <li>Coffee</li>
   <li>Tea</li>
</ol>

<ol>
   <li start="60">Coffee</li>
   <li>Tea</li>
</ol>
