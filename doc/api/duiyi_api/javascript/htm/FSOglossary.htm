Microsoft(R) Scripting 库 - FileSystemObject
FileSystemObject 词汇 		 Scripting 运行时库参考 

 
数组
一组具有相同数据类型，有序索引元素。数组的每个元素有唯一的标识索引号。改变数组的一个元素并不影响其他元素。

 
集合
包含一组相关对象的对象。只要集合中发生了变化，则对象在集合中的位置就可能改变；因而，任何特定的对象在集合中的位置都可能发生了改变。

 
运行时间错误
在代码运行时发生的错误。当语句试图执行一个非法操作时，就会产生一个运行时间错误。

 
字符串表达式
值等于一个连续字符序列的任何表达式。字符串表达式的元素，可以包括返回值是字符串、字符串文字、字符串常数或字符串变量的函数。

 
类型库
在另一个文件中的文件或组件，它含有对被显露对象、属性和方法的标准描述。
