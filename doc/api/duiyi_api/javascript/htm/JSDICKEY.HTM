描述
在 Dictionary 对象中设置一个 key。
语法
object.Key(key) = newkey

Key 属性包括下面这些部分：

部分 描述
object 必选项。总是一个 Dictionary 对象的名称。
key 必选项。被改变的 key 值。
newkey 必选项。替换所指定的 key 的新值。

说明
如果在改变一个 key 时没有发现该 key，那么将创建一个新的 key 并且其相关联的 item 被设置为空。

下面这段代码说明了 Key 属性的用法： 

var d;
d = new ActiveXObject("Scripting.Dictionary");

function AddStuff()
{
  var a;
  d.Add("a", "Athens");
  d.Add("b", "Belgrade");
  d.Add("c", "Cairo");
}

function ChangeKey(oldkey, newkey)
{
  var s;
  d.Key("c") = "Ca";
  s = "Key " + oldkey + " changed to " + newkey;
  return(s);
}
