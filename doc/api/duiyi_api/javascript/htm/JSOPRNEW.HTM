new 运算符
  语言参考

请参阅

描述
创建一个新对象。
语法

new constructor[(arguments)]

constructor 参数调用对象的构造函数。如果构造函数没有参数，则可以省略圆括号。

说明
new 运算符执行下面的任务：
  1. 创建一个没有成员的对象。
  2. 为那个对象调用构造函数，传递一个指针给新创建的对象作为 this 指针。
然后构造函数根据传递给它的参数初始化该对象。

下面这些是有效的 new 运算符的用法例子： 

my_object = new Object;
my_array = new Array();
my_date = new Date("Jan 5 1996");
