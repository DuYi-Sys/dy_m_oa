AvailableSpace 属性		  Scripting 运行时库参考 
版本 3  

请参阅                  应用于

描述
返回在所指定的驱动器或网络共享上可用的空间的大小。
语法
object.AvailableSpace

其中 object 总是一个 Drive 对象。

说明
在典型情况下，由 AvailableSpace 属性所返回的值与 FreeSpace 属性所返回的值是一样的。但是对于支持 quotas 的两个计算机系统之间返回值可能会有所不同。

下面这段代码说明了 AvailableSpace 属性的用法： 

function ShowAvailableSpace(drvPath)
{
  var fso, d, s;
  fso = new ActiveXObject("Scripting.FileSystemObject");
  d = fso.GetDrive(fso.GetDriveName(drvPath));
  s = "Drive " + drvPath.toUpperCase() + " - ";
  s += d.VolumeName + "<br>";
  s += "Available Space: " + d.AvailableSpace/1024 + " Kbytes";
  return(s);
}
