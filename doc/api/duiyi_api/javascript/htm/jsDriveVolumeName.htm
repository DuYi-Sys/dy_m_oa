VolumeName 属性		  SCripting 运行时库参考 
版本 3 

请参阅                  应用于

描述
设置或返回指定驱动器的卷名。 读/写。
语法
object.VolumeName [= newname]

VolumeName 属性包括以下部分：

部分 描述
object 必选项。 总是为 Drive 对象的名称。
newname 可选项。 如果提供了这个部分,那么 newname 就将成为指定的 object 的新名称。

说明
下面的代码说明了 VolumeName 属性的用法： 
function SpaceReport(drvPath)
{
  var fso, d, s;
  fso = new ActiveXObject("Scripting.FileSystemObject");
  d = fso.GetDrive(fso.GetDriveName(drvPath));
  s = "Drive " + drvPath + " - ";
  s += d.VolumeName + "<br>";
  s += "Total Space: "+ d.TotalSize/1024 + " Kbytes <br>";
  s += "Free Space:  " + d.FreeSpace/1024 + " Kbytes";
  return(s);
}
