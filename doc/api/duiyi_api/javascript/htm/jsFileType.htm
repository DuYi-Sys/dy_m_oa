描述
返回关于文件或文件夹类型的信息。 例如，对于以 .TXT 结尾的文件将返回“文本文档”。
语法
object.Type

object 应为 File 或 Folder 对象。

说明
下面的代码说明了利用 Type 属性返回文件夹属性的方法。 在这个例子中，可以尝试一下向过程提供回收站或其他特殊文件夹的路径。 
function ShowFileType(filespec)
{
  var fso, f, s;
  fso = new ActiveXObject("Scripting.FileSystemObject");
  if (fso.FolderExists(filespec))
    f = fso.GetFolder(filespec);
  else if (fso.FileExists(filespec))
    f = fso.GetFile(filespec);
  else
    s = "File or Folder does not exist.";
  s = f.Name + " is a " + f.Type; 
  return(s);
}
