concat 方法 (数组)		  语言参考 
版本 3 

请参阅                  应用于

描述
返回一个新数组，这个新数组是由两个数组组合而成的。
语法
array1.concat(array2)

concat 方法的语法包括下面这些部分：

部分 描述
array1 必选项。一个与 array2 进行连接的 Array 对象。
array2 必选项。一个连接到 array1 末尾的 Array 对象。

说明
concat 方法返回一个 Array 对象，其中包含了 array1 和 array2 的连接。

如果一个对象引用被从 array1 或 array2 复制到结果中，那么结果中的对象引用仍然指向同一个对象。对该对象的改变将同时反映在两个数组中。
