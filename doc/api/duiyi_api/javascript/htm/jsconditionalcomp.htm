描述
允许使用 JScript 语言的新特性，同时保持与不支持这些新功能的浏览器之间的兼容性。
说明
通过在注释外部使用 @cc_on 语句或者使用 @if 或 @set 语句，就可以激活条件编译。条件编译的一些典型用法包括了使用 JScript 中的一些新功能，在脚本中嵌入对调试的支持，以及跟踪代码的执行等。

我们极力推荐将条件编译代码放置在注释中： 

/*@cc_on @*/
/*@if (@_jscript_version == 4)
   alert("JScript version 4");
   @else @*/
   alert("You need a more recent script engine.");
/*@end @*/
本例中使用了一种特殊的注释定界符，这种定界符只有在 @cc_on 条件编译语句激活的情况下才可以使用。不支持条件编译的 Scripting 引擎只能看到一条需要新的 scripting 引擎的提示消息。
