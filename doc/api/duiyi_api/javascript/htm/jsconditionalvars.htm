条件编译可以使用下面这些预定义的变量。如果一个变量值不是 true，那么它就是没有经过定义的，而且如果使用这种变量，它就相当于 NaN。

变量 描述
@_win32 true 如果在 Win32 系统上运行。
@_win16 true 如果在 Win16 系统上运行。
@_mac true 如果在 Apple Macintosh 系统上运行。
@_alpha true 如果在 DEC Alpha 处理器上运行。
@_x86 true 如果在 Intel 处理器上运行。
@_mc680x0 true 如果在 Motorola 680x0 处理器上运行。
@_PowerPC true 如果在 Motorola PowerPC 处理器上运行。
@_jscript 总是为 true。
@_jscript_build 包含 JScript scripting 引擎的编译序号。
@_jscript_version 包含以 major.minor 格式表示的 JScript 版本号。
