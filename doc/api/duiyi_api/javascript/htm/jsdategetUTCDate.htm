getUTCDate 方法		  语言参考 
版本 3  

请参阅                  应用于

描述
返回 Date 对象中用全球标准时间 (UTC)表示的日期。
语法
objDate.getUTCDate()
说明
要获取用本地时间表示的日期，请使用 getDate 方法。

返回值是一个处于 1 到 31 之间的整数值，这个整数代表了 Date 对象中的日期值。

下面这个例子说明了 getUTCDate 方法的用法： 

function UTCDateDemo()
{
  var d, s = "今天 UTC 日期是: ";
  d = new Date();
  s += (d.getUTCMonth() + 1) + "/";
  s += d.getUTCDate() + "/";
  s += d.getUTCFullYear();
  return(s);
}
