getUTCFullYear 方法		  语言参考 
版本 3  

请参阅                  应用于

描述
返回 Date 对象中用全球标准时间 (UTC)表示的年份值。
语法
objDate.getUTCFullYear()
说明
要获取用 本地时间表示的年份值，请使用 getFullYear 方法。

getUTCFullYear 方法以绝对数字的形式返回年份值。这样可以避免出现 2000 年问题，从而不会将 2000 年1月1日以后的日期与 1900 年1月1日以后的日期混淆起来。

下面这个例子说明了 getUTCFullYear 方法的用法： 

function UTCDateDemo()
{
  var d, s = "今天 UTC 日期是: ";
  d = new Date();
  s += (d.getUTCMonth() + 1) + "/";
  s += d.getUTCDate() + "/";
  s += d.getUTCFullYear();
  return(s);
}
