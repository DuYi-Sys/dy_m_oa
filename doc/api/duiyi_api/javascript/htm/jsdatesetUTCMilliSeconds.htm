setUTCMilliseconds 方法		  语言参考 
版本 3  

请参阅                  应用于

描述
设置 Date 对象中用全球标准时间 (UTC)表示的毫秒值。
语法
objDate.setUTCMilliseconds(numMilli)

其中 numMilli 参数是一个等于毫秒值的数值。

说明
要设置用 本地时间表示的毫秒值，请使用 setMilliseconds 方法。

如果 numMilli 的值大于 999 或者是负数，那么秒钟数（以及分钟、小时等等，如果需要的话）将增加一个适当的数量。

下面这个例子说明了 setUTCMilliseconds 方法的用法： 

function SetUTCMSecDemo(nmsec)
{  
  var d, s;
  var sep = ":";
  d = new Date();
  d.setUTCMilliseconds(nmsec);
  s = "Current setting is ";
  s += d.toUTCString() + sep + d.getUTCMilliseconds();
  return(s);
}
