setUTCSeconds 方法		  语言参考 
版本 3  

请参阅                  应用于

描述
设置 Date 对象中用全球标准时间 (UTC)表示的秒值。
语法
objDate.setUTCSeconds(numSeconds[, numMilli])

setUTCSeconds 方法的语法包括下面这些部分：

部分 描述
numSeconds 必选项。等于秒值的数值。
numMilli 可选项。等于毫秒值的数值。

说明
如果没有指定可选项，那么所有使用可选项的 set 方法都将使用从相应的 get 方法返回的数值作为可选项的数值。例如，如果 numMonth 参数是可选的，但是没有被指定，JScript 将使用从 getMonth 方法返回的值作为该参数的值。

要设置用本地时间表示的秒钟值，请使用 setSeconds 方法。

如果参数的的值超出其范围或者是负数，其他被保存的值都将相应地得到修改。例如，如果所保存的日期是 "Jan 5, 1996 00:00:00"，且调用了 setSeconds(150) 方法，那么日期将被改变为 "Jan 5, 1996 00:02:30.00."

下面这个例子说明了 setSeconds 方法的用法： 

function SetUTCSecondsDemo(nsec, nmsec)
{  
  var d, s;
  var sep = ":";
  d = new Date();
  d.setUTCSeconds(nsec, nmsec);
  s = "Current setting is ";
  s += d.toUTCString() + sep + d.getUTCMilliseconds();
  return(s);
}
