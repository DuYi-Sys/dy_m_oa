setDate 方法 		 语言参考 
版本第 1  

请参阅                  应用于

描述
设置 Date 对象中用本地时间表示的数字日期。
语法
objDate.setDate(numDate)

其中 numDate 参数是一个等于数字日期的数值。

说明
要设置用全球标准时间 (UTC)表示的日期值，请使用 setUTCDate 方法。

如果 numDate 的值大于 Date 对象中所保存的月份的天数或者是负数。那么日期将被设置为由 numDate 减去所保存月份中天数而得到的日期。例如，如果所保存的日期是 1996 年 1 月 5 日，并且调用了方法 setUTCDate(32)，那么日期将被改变为 1996 年 2 月 1 日。负数的处理方法与此相似。

下面这个例子说明了 setDate 方法的用法： 

function SetDateDemo(newdate)
{
  var d, s;
  d = new Date();
  d.setDate(newdate);
  s = "Current setting is ";
  s += d.toLocaleString(); 
  return(s);
}
