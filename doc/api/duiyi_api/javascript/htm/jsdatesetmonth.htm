setMonth 方法 		 语言参考 
版本 1  

请参阅                  应用于

描述
设置 Date 对象中用 本地时间表示的月份值。
语法
objDate.setMonth(numMonth[, dateVal])

setMonth 方法的语法有下面这些部分：

部分 描述
numMonth 必选项。一个等于月份值的数值。
dateVal 可选项。一个代表日期的数值。如果没有提供此参数，那么将使用通过调用 getDate 方法而得到的数值。

说明
要设置用全球标准时间 (UTC)表示的月份值，请使用 setUTCMonth 方法。

如果 numMonth 的值大于 11 (月份 0 表示一月)或者是一个负数，那么所保存的年份将相应地得到改变。例如，如果所保存的日期是 "Jan 5, 1996" 并且调用了 setMonth(14) 方法，那么该日期就被改变为 "Mar 5, 1997."

下面这个例子说明了 setMonth 方法的用法： 

function SetMonthDemo(newmonth)
{  
  var d, s;
  d = new Date();
  d.setMonth(newmonth);
  s = "Current setting is ";
  s += d.toLocaleString(); 
  return(s);
}
