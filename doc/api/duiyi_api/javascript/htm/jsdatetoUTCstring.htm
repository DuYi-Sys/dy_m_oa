描述
返回一个已被转换为字符串的，用全球标准时间 (UTC)表示的日期。
语法
objDate.toUTCString()
说明
toUTCString 方法返回一个 String 对象，此对象中包含了使用 UTC 惯例以一种方便易读的形式进行格式化的日期。

下面这个例子说明了 toUTCString 方法的用法： 

function toUTCStrDemo()
{  
  var d, s;
  d = new Date();
  s = "Current setting is ";
  s += d.toUTCString(); 
  return(s);
}
