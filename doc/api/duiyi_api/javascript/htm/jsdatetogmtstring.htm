toGMTString 方法 		 语言参考 
版本 1  

请参阅                  应用于

描述
返回一个日期，该日期用格林威治标准时间 (GMT) 表示并已被转换为字符串。
语法
objDate.toGMTString()
说明
toGMTString 方法已经过时，之所以仍然提供这个方法，只是为了提供向后的兼容性。推荐改用 toUTCString 方法。

toGMTString 方法返回一个 String 对象，此对象中包含了按照 GMT 惯例进行格式化的日期。返回值的格式如下："05 Jan 1996 00:00:00 GMT."

下面这个例子说明了 toGMTString 方法的用法： 

function toGMTStrDemo()
{  
  var d, s;
  d = new Date();
  s = "Current setting is ";
  s += d.toGMTString(); 
  return(s);
}
