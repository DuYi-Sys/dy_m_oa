atEnd 方法		  语言参考 
版本 3 

请参阅                  应用于

描述
返回一个 Boolean 值，指明枚举算子是否位于集合的末尾。
语法
myEnum.atEnd( )

myEnum 参数是任意 Enumerator 对象。

返回值
如果当前项是集合中的最后一个，或者集合为空，或者当前项没有定义,那么 atEnd 方法将返回 true 。否则返回 false 。
说明
在下面的代码中，使用了 atEnd 方法来决定是否到达了一个驱动器列表的末尾： 
function ShowDriveList()
{
  var fso, s, n, e, x;
  fso = new ActiveXObject("Scripting.FileSystemObject");
  e = new Enumerator(fso.Drives);
  s = "";
  for (; !e.atEnd(); e.moveNext())
  {
    x = e.item();
    s = s + x.DriveLetter;
    s += " - ";
    if (x.DriveType == 3)
      n = x.ShareName;
    else if (x.IsReady)
      n = x.VolumeName;
    else
      n = "[驱动器未就绪]";
    s +=  n + "<br>";
  }
  return(s);
}
