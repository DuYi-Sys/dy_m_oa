描述
返回由字符串转换得到的浮点数。
语法
parseFloat(numstring)

numstring 参数是包含浮点数的字符串。

注释
parseFloat 方法返回与 numstring 中保存的数相等的数字表示。如果 numstring 的前缀不能解释为浮点数，则返回 NaN （而不是数字）。
parseFloat("abc")    // 返回 NaN。
parseFloat("1.2abc") // 返回 1.2。
可以用 isNaN 方法检测 NaN。
