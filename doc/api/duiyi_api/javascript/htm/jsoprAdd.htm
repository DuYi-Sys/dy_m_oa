描述
用于求两个数字的和或进行字符串连接。
语法
result = expression1 + expression2

+ 运算符语法有以下部分：

部分 描述
result 任何变量。
expression1 任何表达式。
expression2 任何表达式。

说明
表达式中加下划线的子类型决定 + 运算符的操作。

如果 那么
两个表达式都是数字或 Boolean 量 加。
两个表达式都是字符串 连接。
一个表达式是数字而另一个是字符串 连接。

有关 + 运算符何时会产生运行时错误的信息，请参见运算符操作表。
