描述
对两个表达式执行逻辑“或”。
语法
result = expression1 || expression2

|| 运算符的语法组成部分如下：

部分 描述
result 任何变量。
expression1 任何表达式。
expression2 任何表达式。

说明
如果两个表达式中有一个或两个等于 True，则 result 是 True。下面的表举例说明了 result 是如何被决定的：

如果 expression1 是 且 expression2 是 则 result 是
True True True
True False True
False True True
False False False

关于 && 运算符何时产生运行时错误的信息，请参阅运算符操作表。

JScript 使用下面的规则来把非 Boolean 值转换为 Boolean 值：

  • 所有对象都被认为是 true。
  • 字符串当且仅当为空时才被认为是 false。
  • null 和 undefined 被认为是 false。
  • 数字当且仅当为 0 时才是 false。
