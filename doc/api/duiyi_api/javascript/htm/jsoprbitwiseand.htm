描述
对两个表达式执行按位“与”。
语法
result = expression1 & expression2

& 运算符的语法组成部分如下：

部分 描述
result 任何变量。
expression1 任何表达式。
expression2 任何表达式。

说明
& 运算符查看两个表达式的二进制表示法的值，并执行按位“与”操作。该操作的结果如下所示： 
0101    (expression1)
1100    (expression2)
----
0100    (result)
任何时候，只要两个表达式的某位都为 1，则结果的该位为 1。否则，结果的该位为 0。

关于 & 运算符何时产生运行时错误的信息，请参阅运算符操作表。
