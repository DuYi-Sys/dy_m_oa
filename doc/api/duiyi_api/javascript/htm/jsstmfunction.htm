描述
声明一个新的函数。
语法
function functionname([argument1 [, argument2 [, ...argumentn]]])
{
   statements
}

function 语句的语法组成如下：

部分 描述
functionname 函数名。
argument1...argumentn 可选的、函数理解的参数列表、各个参数之间用逗号分开。
statements 一个或多个 JScript 语句。

说明
使用 function 语句来声明一个稍后要使用的函数。在脚本的其他地方调用该函数前，statements 中包含的代码不被执行。

下面示例说明了 function 语句的用法： 

function myfunction(arg1, arg2)
{
  var r;
  r = arg1 * arg2;
  return(r);
}
注意  在调用函数时，确保总是包括了圆括号以及任何需要的参数。不带圆括号调用函数将导致返回函数的文本，而不是函数的结果。
