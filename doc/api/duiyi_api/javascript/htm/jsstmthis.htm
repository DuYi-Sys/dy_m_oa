描述
指当前对象。
语法
this.property
说明
this 关键字通常在对象的 构造函数中使用，用来引用对象。在下面示例中，this 指的是新创建的 Car 对象，并给三个属性赋值： 
function Car(color, make, model)
{
  this.color = color;
  this.make = make;
  this.model = model;
}
对于 JScript 的客户版本，如果在其他所有对象的上下文之外使用 this，则它指的是 window 对象。
