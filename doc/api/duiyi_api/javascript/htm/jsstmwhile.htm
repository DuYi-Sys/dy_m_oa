描述
执行一个语句，直到指定的条件为 false。
语法
while (expression)
   statement

while 语句的语法组成如下：

部分 描述
expression Boolean 表达式，在循环的每次迭代前被检查。 如果 expression 是 true，则执行循环。如果 expression 是 false，则结束循环。
statement expression 是 true 时要执行的语句。可以是复合语句。

说明
while 语句在循环第一次被执行前检查 expression。如果 expression 此次是 false，则该循环一次都不被执行。

下面示例说明了 while 语句的用法： 

function BreakTest(breakpoint)
{
   var i = 0;
   while (i < 100)
   {
   if (i == breakpoint)
     break;
     i++;
   }
   return(i);
}
