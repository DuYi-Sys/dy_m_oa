描述
把 HTML <BIG> 标记放置在 String 对象中的文本两端。
语法
strVariable.big( )
"String Literal".big( )
说明
下面示例说明了 big 方法是如何工作的： 
var strVariable = "This is a string object";
strVariable = strVariable.big( );
在执行完最后一个语句后，strVariable 的值为： 
<BIG>This is a string object</BIG>
不检查该标记是否已经被应用于该字符串了。
