描述
把 HTML <B> 标记放置在 String 对象中的文本两端。
语法
strVariable.bold( )
"String Literal".bold( )
说明
下面的示例说明了 bold 方法是如何工作的： 
var strVariable = "This is a string object";
strVariable = strVariable.bold( );
最后一条语句执行完后，strVariable 的值是： 
<B>This is a string object</B>
不检查该标记是否已经被应用于该字符串了。
