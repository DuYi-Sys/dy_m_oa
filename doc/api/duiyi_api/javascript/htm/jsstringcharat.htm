charAt 方法 		 语言参考 
版本 1 

请参阅                  应用于

描述
返回指定索引位置处的字符。
语法
strVariable.charAt(index)
"String Literal".charAt(index)

index 参数是想得到的字符的基于零的索引。有效值是 0 与字符串长度减 1 之间的值。

说明
charAt 方法返回一个字符值，该字符位于指定索引位置。字符串中的第一个字符的索引为 0，第二个的索引为 1，等等。超出有效范围的索引值返回 undefined。

下面的示例说明了 charAt 方法的用法：

function charAtTest(n)
{
  var str = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
  var s;
  s = str.charAt(n - 1);
  return(s);
}
