concat 方法（String）		  语言参考 
版本 3 

请参阅                  应用于

描述
返回一个 String 对象，该对象包含了两个提供的字符串的连接。
语法
string1.concat(string2)

concat 方法的语法组成如下：

部分 描述
string1 必选项。要和 string2 连接的 String 对象或文字。
string2 必选项。要连接到 string1 尾部的 String 对象或文字。

说明
concat 方法的结果等于： result = string1 + string2.

下面的示例说明了 concat 方法的用法：

function concatDemo()
{
  var str1 = "ABCDEFGHIJKLM"
  var str2 = "NOPQRSTUVWXYZ";
  var s = str1.concat(str2);
  // 返回连接好的字符串。
  return(s);
}
