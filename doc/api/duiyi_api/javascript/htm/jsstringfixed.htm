描述
把 HTML <TT> 标记放置在 String 对象中的文本两端。
语法
strVariable.fixed( )
"String Literal".fixed( )
说明
下面的示例说明了 fixed 方法如何工作： 
var strVariable = "This is a string object";
strVariable = strVariable.fixed( );
最后一条语句执行完后，strVariable 的值是： 
<TT>This is a string object</TT>
不检查该标记是否已经被应用于该字符串了。
