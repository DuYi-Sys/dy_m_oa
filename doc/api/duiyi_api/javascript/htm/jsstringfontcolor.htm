fontcolor 方法 		 语言参考

请参阅                  应用于

描述
把带有 COLOR 属性的一个 HTML <FONT> 标记放置在 String 对象中的文本两端。
语法
strVariable.fontcolor(colorval)
"String Literal".fontcolor(colorval)

colorval 参数是一个包含一颜色值的字符串。可以是颜色的十六进制值，或预先定义好的颜色名字。

说明
下面的示例说明了 fontcolor 方法： 
var strVariable = "This is a string";
strVariable = strVariable.fontcolor("red");
最后一条语句执行完后，strVariable 的值是： 
<FONT COLOR="RED">This is a string</FONT>
有效的预定义颜色名取决于 JScript 主机（浏览器，服务器，等等）。它们也可能由主机版本的不同而不同。请查阅主机的说明书来获得更多的信息。

不检查该标记是否已经被应用于该字符串了。
