描述
把 HTML <I> 标记放置在 String 对象中的文本两端。
语法
strVariable.italics( )
"String Literal".italics( )
说明
下面的示例说明了 italics 方法是如何工作的： 
var strVariable = "This is a string";
strVariable = strVariable.italics( );
最后一条语句执行完后，strVariable 的值是： 
<I>This is a string</I>
不检查该标记是否已经被应用于该字符串了。
