search 方法		  语言参考 
版本 3 

请参阅                  应用于

描述
返回与正则表达式查找内容匹配的第一个子字符串的位置。
语法
stringObj.search(rgexp)

search 方法的语法包括下述部分：

部分 描述
stringObj 必选项。进行查找的 String 对象或文字。
rgexp 必选项。包含要查找的模式的正则表达式对象。

说明
search 方法指明是否存在相应的匹配。如果找到一个匹配，search 方法将返回一个整数值，指明这个匹配距离字符串开始的偏移位置。如果没有找到匹配，则返回 -1。要获得进一步的信息，请使用 match 方法。

下面的示例演示了 search 方法的用法：

function SearchDemo()
{
  var r, re;
  var s = "The quick brown fox jumped over the lazy yellow dog.";
  re = /fox/i;
  r = s.search(re);
  return(r);
}
