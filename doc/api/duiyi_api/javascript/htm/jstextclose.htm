Close 方法		  Scripting 运行时库参考 
版本 2 

请参阅                  应用于

描述
关闭打开的 TextStream 文件。

语法
object.Close( );

object 总是 TextStream 对象的名称。

说明
下面的示例演示了Close 方法的用法:

var fso;
fso = new ActiveXObject("Scripting.FileSystemObject");
a = fso.CreateTextFile("c:\\testfile.txt", true);
a.WriteLine("This is a test.");
a.Close();
